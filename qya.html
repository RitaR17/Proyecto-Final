PREGUNTAS INTRODUCTORIAS Y sus respuestas

¿Qué es un framework?


Es una estructura base utilizada como punto de partida para elaborar un proyecto con objetivos específicos

¿Qué es Bootstrap 5 y cuál es su origen?


Bootstrap es un marco CSS gratuito y de código abierto dirigido al desarrollo web front-end receptivo y móvil.originalmente llamado Blueprint de Twitter, fue desarrollado por Mark Otto y Jacob Thornton de Twitter, como un marco de trabajo (framework) para fomentar la consistencia entre las herramientas internas.

¿Qué es diseño responsive en páginas web?


Esta es una técnica de diseño web que busca la correcta visualización de una misma página en distintos dispositivos, como vista en tableta o un teléfono.

¿Qué son las Github pages?


GitHub es un proveedor de alojamiento de Internet para el desarrollo de software y el control de versiones.

¿Cuál es la diferencia entre elemento y componente web?


Los componentes de una página web son: imágenes, texto, audio, video, animaciones y otros contenidos.
Los elementos de una pagina web son:popágina Paleta de colores de la página,presencia en las redes sociales,información relevante sobre tu negocio,Página web responsive.

¿Cuál es la importancia de documentar o comentar código?

Ya que no solo nos sirve a nosotros como una guía y apoyo,también sirve al momento de trabajar en equipo o cuando compartimos código

¿Por qué existe la sección de atribución en un sitio web?


Es parte de nuestra ética como desarrolladores de contenido, ya que así como nos gustaría que nos dieran credito y no solo "robar" la información, de nuestra parte es importante conocer otra maneras y encontar media que nos sirva como aprender.
